Dataset Structure

Component Path Description
Background images images/backgrounds/ High-resolution scene images with target product regions
Product images images/products/ Replacement product images for compositing
Masks masks/ Freeform, bounding box, and dimension-aware masks per example

Metadata dataset_metadata.json Source URLs, licensing information, and quantitative metrics
PDF summaries additional_results_*.pdf Full visual results for all 58 benchmark examples

All masks and model outputs are bundled and render directly in the interactive viewers above. Generated assets (without external source URLs) are explicitly marked in the metadata.
